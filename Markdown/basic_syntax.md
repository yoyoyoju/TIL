# Basic syntax

### Headers

```markdown
# This is an <h1> tag
## This is an <h2> tag
```

### Emphasis

```markdown
*italic*
_italic_

**bold**
__bold__

_italic **bold**_
```
the last line:
_italic **bold**_

### Lists

#### Unordered

```markdown
* Item 1
* Item 2
    * Item 2a
        * Item 2a-1
```
* Item 1
* Item 2
    * Item 2a
        * Item 2a-1


#### Ordered

```markdown
1. Item 1
1. Item 2
    1. Item 2a
```
1. Item 1
1. Item 2
    1. Item 2a


### Images
```markdown
![Little Dog](example_picture.png "Dog")
Format: ![AltText](url)
```
![Little Dog](example_picture.png "Dog")
Format: ![AltText](url)


### Links
```markdown
http://github.com - automatic!
[GitHub](http://github.com)
```

### Blockquotes

```markdown
I want to quote:

> that
```
