# GitHub Flavored Markdown

### Syntax highlighting

#### without highlight

1. indent your code by four spaces:
```markdown
    def foo():
        if not bar:
            return True
```

2. use fenced code
`````
```
def foo():
    if not bar:
        return True
```
`````

#### with highlight

`````markdown
```python
def foo():
    if not bar:
        return True
```
`````
